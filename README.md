# Viyaasan Thanabalasingam — Portfolio

Modern single-page portfolio built with **React + Vite**, served via **nginx** in Docker.

## Local Development

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # production build
```

## Docker

```bash
# Build
docker build -t viyaasan-portfolio .

# Run
docker run -p 8080:80 viyaasan-portfolio
# Open http://localhost:8080
```
